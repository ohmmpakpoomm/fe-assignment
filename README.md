## 🛠️ Tech Stack

- **Core**: React 19, TypeScript, Vite
- **State Management**: Zustand
- **Data Handling**: TanStack Table v8, TanStack Virtual v3
- **Styling**: Tailwind CSS v4
- **Feedback**: React Toastify

## 📥 Getting Started (Fresh Machine Setup)

Follow these steps to get the project running on your local machine:

### Prerequisites

- **Node.js**: Version 18.x or higher is recommended.
- **npm**: Usually comes bundled with Node.js.

### Installation

1. **Clone the project** (if not already done):
   ```bash
   git clone https://github.com/ohmmpakpoomm/fe-assignment.git
   cd fe-assignment
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:5173`.


## 🏗️ Project Structure

- `/src/components`: UI components organized by feature (layout, orders).
- `/src/store`: Centralized state management using Zustand.
- `/src/lib`: Core business logic, including allocation algorithms and utility functions.
- `/src/data`: Mock data and initial state configurations.
- `/src/types`: TypeScript interfaces and type definitions.
